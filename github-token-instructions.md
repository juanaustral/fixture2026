# Para darme un token de GitHub

1. Abrí este link:  
   👉 **https://github.com/settings/tokens/new**

2. Dale un nombre (ej: "akira-mundial")

3. En **Expiration** poné "No expiration" o "7 days"

4. Tildá SOLO este scope:  
   ☑️ **`repo`** (Full control of private repositories)

5. Click **Generate token**

6. Copiá el token (empieza con `ghp_...`) y pegámelo acá
